# Interview Preparation System Frontend

This version refreshes the UI with a cleaner white-space-heavy design while keeping the backend API contract unchanged.

## Run locally

```bash
copy .env.example .env
npm install
npm run dev
```

## Assets included

- Recruit Riders logo
- favicon
- company logo strip icons
- office and team images for the company section

## Environment

Set `VITE_API_BASE_URL` in `.env`.
